What LoopBench measures

Closed loops, not prompts

You submit an LSS YAML loop spec. LoopBench runs it through LoopGym SimEnv on fixed instances and computes Success@k and observed LES across eight categories.

Reproducible by design

Five seeds per task. No API keys on v0.1. Every row links to a spec and a one-line loopbench run repro command.

Community-owned scoreboard

External submitters get leaderboard credit. Maintainer baselines are labeled. Human review on every merge.
